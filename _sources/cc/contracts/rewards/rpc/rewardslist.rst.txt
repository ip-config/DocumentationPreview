***********
rewardslist
***********

Lists all the rewards contracts available
=========================================

This command shows you the list of availiable contracts. If you want to see detailed info, please use rewardsinfo call.

Example Command:
----------------

.. code-block:: shell

    ./komodo-cli -ac_name=ATEST rewardslist

Output:

::

    [
      "6a9f3a0a7a570bce1022a13f5f9b613d1431fa11dba9a09e10fe54b6998ff8d5"
    ]
